## 侧边栏菜单结构

### Guest 模式
```
📅 研讨会日程
   └─ 查看日程
   └─ 场次可用性
🔐 登录
```

### Student 模式
```
📅 研讨会日程
   └─ 查看日程
📝 我的注册
   └─ 注册研讨会
   └─ 注册状态
📤 材料上传
   └─ 上传演示文稿
👤 个人中心
🚪 登出
```

### Evaluator 模式
```
📅 研讨会日程
   └─ 查看日程
📋 评估任务
   └─ 待评估列表
   └─ 已完成评估
✍️ 评分面板
👤 个人中心
🚪 登出
```

### Coordinator 模式
```
📊 仪表盘
👥 用户管理
   └─ 学生管理
   └─ 评估员管理
📅 场次管理
   └─ 创建场次
   └─ 场次列表
📝 注册管理
   └─ 注册审批
   └─ 分配管理
🎯 评估管理
   └─ 评估员分配
   └─ 评估查看
🏆 奖项管理
   └─ 奖项计算
   └─ 颁奖典礼
📈 报告中心
   └─ 统计报告
   └─ 导出数据
⚙️ 系统设置
🚪 登出
```
